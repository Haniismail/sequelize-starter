

> Sequelize Node.js starter containing roles auth models controllers and uppload middleware all ready to use , developed using Typscript, Node.js, Express.js, Sequelize and MysSQL.

## Prerequisites

- npm >=5.5.0
- node >=16.13.1

## Install

```
npm install
```

## Run tests

```
npm run test
```
